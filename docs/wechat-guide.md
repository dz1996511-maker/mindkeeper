# 微信接入指南

2026 年腾讯通过 OpenClaw（iLink 协议）开放了微信个人号的 Bot API，这是官方合规渠道。

## 前提条件

- iOS 微信（当前仅支持 iOS，安卓尚未开放）
- 一个微信账号

## 接入方式

### 方式一：通过 AstrBot WebUI 配置（推荐）

1. 在 WebUI 中进入「平台适配器」设置
2. 添加 WeChat 适配器（Gewechat 或 OpenClaw 类型）
3. 按照提示扫描二维码登录
4. 登录成功后会显示「已连接」

### 方式二：使用 my-wechat-ai（快速体验）

在**本地电脑**上运行：

```bash
npx my-wechat-ai
```

扫码即可体验微信 AI 对话。

## 注意事项

- 一个微信只能绑定一个 Bot 实例
- 不能主动推送消息（需用户先发消息触发）
- 暂不支持群聊消息
- 建议使用小号或备用号

## 故障排查

| 问题 | 解决方法 |
|------|---------|
| 二维码过期 | 在 WebUI 中重新生成 |
| 无法连接 | 检查服务器防火墙端口是否开放 |
| 消息收不到 | 检查容器日志: `docker compose logs -f` |
