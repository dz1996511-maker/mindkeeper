# 飞书接入指南

## 创建飞书机器人

### 步骤 1: 创建应用

1. 打开 [飞书开放平台](https://open.feishu.cn/)
2. 登录后进入「开发者后台」
3. 点击「创建应用」→「企业自建应用」
4. 填写应用名称等信息

### 步骤 2: 配置机器人

1. 在应用详情页，点击「添加能力」→「机器人」
2. 开启机器人功能
3. 在「事件订阅」中添加事件:
   - `im.message.receive_v1`（接收消息）
4. 配置回调地址:
   `http://<你的服务器IP>:6196/webhook/feishu`

### 步骤 3: 获取凭证

1. 进入「凭证与基础信息」
2. 记录 **App ID** 和 **App Secret**

### 步骤 4: 发布应用

1. 在「版本管理与发布」中创建版本
2. 填写版本说明
3. 提交审核（企业自建应用通常是自动通过）
4. 审核通过后，在企业工作台中找到该应用

### 步骤 5: 在 AstrBot 中配置

1. 打开 WebUI → 「平台适配器」
2. 添加 Feishu 适配器
3. 填入 App ID 和 App Secret

## 使用

- 在飞书中打开该应用，直接发消息即可
- 或添加到群聊中 @机器人

## 注意事项

- 飞书 API 有频率限制（企业版 100次/秒）
- Webhook URL 需要使用 HTTP/HTTPS
- 需要配置公网可访问的服务器
