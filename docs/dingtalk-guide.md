# 钉钉接入指南

## 创建钉钉机器人

### 步骤 1: 创建应用

1. 打开 [钉钉开放平台](https://open.dingtalk.com/)
2. 登录后进入「应用开发」→「企业内部应用」
3. 点击「创建应用」，填写应用名称等信息
4. 创建完成后，记录 **AppKey** 和 **AppSecret**

### 步骤 2: 配置机器人

1. 在应用详情页，点击「添加应用能力」→「机器人」
2. 填写机器人名称、描述、图标
3. 消息接收模式选择 **Webhook**
4. 消息接收地址填写:
   `http://<你的服务器IP>:6195/webhook/dingtalk`
5. 开启「企业机器人」或「群机器人」权限

### 步骤 3: 在 AstrBot 中配置

1. 打开 WebUI → 「平台适配器」
2. 添加 DingTalk 适配器
3. 填入 AppKey 和 AppSecret

## 使用

- 在钉钉中向机器人发送消息
- 或把机器人添加到群聊中 @它

## 注意事项

- 钉钉机器人有频率限制（企业版 20次/秒）
- 需要确保服务器可以访问公网
- Webhook URL 需要使用 HTTP/HTTPS
