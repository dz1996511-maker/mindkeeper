# MindKeeper 客户对接指南

## 服务内容

为你部署一套个人 AI 知识库系统，实现：
- 微信/钉钉/飞书发消息 → AI 自动分类 → 结构化保存 → 随时导出

## 你需要准备的

1. **云服务器**（可代购）：
   - 推荐 2核4G，阿里云/腾讯云约 99-199元/年
   - 或者我这边统一托管

2. **LLM API Key**（可代购）：
   - Claude API（推荐，质量高）
   - 或 DeepSeek API（价格低）

3. **IM 平台账号**（自备）：
   - 微信个人号（当前仅支持 iOS）
   - 或钉钉/飞书账号

## 部署流程

| 步骤 | 内容 | 时间 |
|------|------|------|
| 1 | 购买云服务器 | 10分钟 |
| 2 | 部署 Docker + AstrBot | 30分钟 |
| 3 | 配置 LLM API | 10分钟 |
| 4 | 接入 IM 平台 | 20分钟/个 |
| 5 | 启用分类插件 | 5分钟 |
| 6 | 测试验收 | 15分钟 |

**总计：约 1.5 小时**

## 使用方式

- 在微信/钉钉/飞书上正常发消息
- 机器人自动识别有价值的消息并分类保存
- 发送 `/export` 一键导出全部记录
- 发送 `/record <内容>` 手动记录

## 常见问题

**Q: 微信会封号吗？**
A: 使用腾讯官方 OpenClaw 协议，扫码授权，合规渠道，封号风险极低。

**Q: 数据安全吗？**
A: 数据存储在你自己的服务器上，仅 LLM 分类时需要联网调用 API。

**Q: 需要自己维护吗？**
A: Docker 容器自动运行，基本零维护。有问题远程协助。
